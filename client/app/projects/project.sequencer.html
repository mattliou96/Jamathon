Getting Started...


Play the Project

Hit PLAY or 'right-arrow'

Create a new note

Click on an empty note position to insert a new note

Select and/or edit a note

Double click a note to select it. Double Click the same note to unselect a note. Selecting a new note by double click action will also deselect the previous note

Set a notes length

Double click on a note to select it and then click an empty note slot further ahead in the time line to set the notes length

Delete a note

While a note is selected, press the 'delete' key or the 'left-arrow'

Set the Pitch of a note

While a note is selected use keyboard entry from 'Z' to '/' to set the pitch of the notes
